关于在Typora中插入图片上传至github中显示问题

第一步：在终端新建博客文章

在终端利用命令：`hexo new post 文章名`在hexo主题根目录建立markdown文档，这个文档就是一篇博客文章

markdown文档可以再hexo主题的根目录找见

第二步：存放图片

将所需的图片放置在同名的文件夹内

第三步：插入图片

利用代码 来插入图片

第四步：上传至github

将文档保存后，打开终端，进入主题根目录，输入命令 hexo g 后在输入hexo d 进行上传
